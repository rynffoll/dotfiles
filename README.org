* Envy
Ansible scripts for deployment local environment

** Requirements
- ansible
- python2-psutil (for dconf module)

** How to Use
Fedora:
#+BEGIN_SRC shell-script
make install_deps_fedora install_roles all
#+END_SRC

macOS:
#+BEGIN_SRC shell-script
make install_deps_macos install_roles all
#+END_SRC
